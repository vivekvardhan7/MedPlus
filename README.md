## 🧪 Setup & Installation

### 🔧 Prerequisites

- Node.js & npm
- MySQL server running
- (Optional) Git & VSCode

---

### 🖥️ Backend Setup

```bash
cd server
npm install
# Create a `.env` file with your DB credentials
npm start


```
### 🌐 Frontend Setup
```bash

cd client
npm install
npm start

Frontend will run on http://localhost:3000/
```
